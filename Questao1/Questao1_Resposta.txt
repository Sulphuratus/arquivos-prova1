1- Defina o que é uma classe dentro da orientação a objetos. (4.0)

Uma classe na programação orientada a objetos é uma forma de representar qualquer coisa do mundo real, seja esta coisa concreta ou abstrata, em um programa de computador para que se possa simular o seu comportamento e obter os resultados desejados. A representação é feita através da criação de um arquivo da classe onde será desenvolvido o programa desejado. Dentro da classe que representa a entidade desejada, poderão ser criados os tipos, também chamados de objetos, que compartilham características comuns dentro da classe. Essas caracterísitcas são chamadas de atributos e são representados por variáveis que podem ser de diversos tipos, String, Inteira, booleana, char, entre outras. Para representar as ações de um objeto são utilizadas funções, também chamadas de métodos. Desta forma é possível representar uma classe de uma entidade qualquer no programa e através dos objetos especificar os atributos e métodos que cada objeto poderá apresentar. Por exemplo, ao criarmos uma classe Cachorro estamos representando todos as entidades dessa classe que possuem atributos em comum como nome, peso, idade, raça, tipo de pelo, entre outros, e métodos como latir, correr, comer, dormir, entre outros. Porém para cada objeto criado, por exemplo cachorro1, cachorro2, etc... podemos relacionar atributos e métodos diferentes. Assim, pode-se dizer que o objeto em programação orientada a objetos é uma representação de uma classe para um tipo específico.
